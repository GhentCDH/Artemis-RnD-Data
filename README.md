# Artemis-RnD-Data

Compiled IIIF + Allmaps data pipeline for Artemis.

This repository crawls source IIIF collections, mirrors georeferencing annotations, compiles manifests/collections for static hosting, and builds text-search artifacts used by the viewer.

## Stack

- Runtime: Bun
- Language: TypeScript
- Core dependency: `@allmaps/id`

## What It Produces

- `build/index.json`: main dataset index (layers + per-manifest entries)
- `build/collection.json`: top-level IIIF collection
- `build/collections/*.json`: per-layer compiled collections
- `build/manifests/*.json`: compiled manifests
- `build/allmaps/*`: mirrored Allmaps annotations
- `build/Toponyms/index.json`: toponym search index

## Quick Start

```bash
bun install
```

## Commands

```bash
# full data pipeline
bun run crawl

# limited crawl (per source)
bun run crawl10
bun run crawl1

# build toponym search index only
bun run buildSearch
# alias
bun run toponyms
```

## Inputs

- `data/sources/collections.txt`: source IIIF collection URLs
- `data/sources/Toponyms/`: local raw toponym source files (not committed)

Toponym source policy:

- keep raw files local
- keep only `data/sources/Toponyms/README.txt` in git
- run `bun run buildSearch` to regenerate `build/Toponyms/index.json`

## Notes

- `cache/` is persistent between runs
- `build/` outputs are regenerated by pipeline/search builders
- `build/index.json` `index[]` entries can include `centerLon` / `centerLat` for manifest click-to-location in viewer
